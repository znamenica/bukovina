# Bukovina

Bukovina is the Orthodox Christian God-service library

## Installation

Add this line to your application's Gemfile:

    gem 'bukovina'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install bukovina

## Testing

### Migration

    $ RAILS_ENV=cucumber rake db:migrate

### Run tests

    $ rake cucumber

## Usage

### Migration

    $ rake db:migrate

### Import

    $ rake db:import
